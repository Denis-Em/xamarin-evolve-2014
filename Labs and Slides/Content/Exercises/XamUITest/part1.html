XTC 102 - Xamarin.UITest

Exercise 1: Xamarin.UITest

Estimated Time: 20 minutes

Goals

The primary goal of this exercise will be to ensure that you can successfully execute a Xamarin.UITest.

Steps

Examine the binary package

This lab has two existing applications, one for Android and the other for iOS, which create a Task application called TaskyPro. You will be executing these applications using Xamarin.UITest. Both of them can be found in the Content/Exercises/XamUITest/Binaries folder.

Create a Unit Test project

  1. Create a new NUnit project from Xamarin Studio.
  2. Right-click on project and select Add solution folder called Unit Test, select Add and then 'Packages...'
  3. Search for Xamarin.UITest and add that package to your project
  4. Add Xamarin.UITest to your list of references in your text file
  5. Create a new test that connects to the application found in the Content/Exercises/XamUITest/Binaries folder.

Android

var app = ConfigureApp.Android.ApkFile(path).StartApp()

iOS

var app = ConfigureApp.iOS.AppBundle(pathToAppFile).StartApp
  1. Use the app.Repl() method to query and control the user interface
  2. Once you have that, write a unit test that creates a new task and saves it to the application and verifies that the test appears.
  3. Execute the Unit Test using the NUnit test runner.
  4. If you have time, write a unit test that attempts to add an entry, but the user cancels before saving. If you still have time then, write a test where a user deletes an existing entry

Summary
